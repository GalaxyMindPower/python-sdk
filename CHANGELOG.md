# Changelog

All notable changes to **AIR Blackbox Python SDK** will be documented in this file.

This project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.1] — 2026-02-22

- Published to PyPI as `air-blackbox-sdk`
- Fixed package metadata and URLs for PyPI listing

## [0.1.0] — 2026-02-21

- Initial release of the AIR Python SDK
- `air_wrap()` function for zero-code-change OpenAI client instrumentation
- LangChain integration (`air_langchain_llm`)
- CrewAI integration (`air_crewai_llm`)
- OpenAI Agents SDK integration (`air_openai_agents_provider`)
- AutoGen integration (`air_autogen_config`)
- Automatic trace correlation via `x-run-id` header
